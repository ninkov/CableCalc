# Cable Calculator

Уеб приложение за ориентировъчно изчисляване на необходимите метри кабел за електроинсталация по стаи.

## Демо

Приложението е достъпно тук:

https://cablecalc.pages.dev/

## Текущ статус

Проектът е в **Версия 1.1** и е изграден с максимално проста архитектура:

- **React + Vite**
- **JavaScript (.jsx)**
- **Tailwind CSS**
- **localStorage** за локално запазване на данните

Текущата версия е фокусирана върху:

- добавяне на стая чрез форма
- редакция на вече запазена стая
- изчисляване на ориентировъчен метраж кабел
- разделяне на резултата по вид кабел
- записване на стаите локално
- визуализиране на списък със запазени стаи
- показване на общ резултат за всички стаи
- защита при стари или непълни данни в `localStorage`

---

## Основна идея

Потребителят попълва данни за стая, след което приложението:

1. изчислява приблизителния необходим метраж кабел
2. записва стаята
3. позволява редакция при нужда
4. позволява добавяне на следваща стая
5. изчислява общото количество кабел за всички въведени помещения

Това прави приложението подходящо за бързи предварителни сметки при изграждане на ел. инсталации.

## Използвана проста архитектура

```bash
src/
  components/
    RoomForm.jsx
    RoomList.jsx
    SummaryCard.jsx
  utils/
    calculateCable.js
    storage.js
  App.jsx
  main.jsx
  index.css
```

### Описание на файловете

#### `App.jsx`
Главен компонент, който:
- пази всички стаи в state
- зарежда стаите от `localStorage`
- записва стаите в `localStorage`
- добавя нова стая
- редактира съществуваща стая
- изтрива стая
- изчиства всички стаи
- обобщава резултатите

#### `components/RoomForm.jsx`
Форма за:
- добавяне на нова стая
- редакция на съществуваща стая

Поддържа:
- попълване на всички основни полета
- зареждане на данните при редакция
- отказ от редакция

#### `components/RoomList.jsx`
Показва всички записани стаи и позволява:
- редакция
- изтриване

За всяка стая визуализира разбивка по:
- осветление
- контакти
- консуматори
- резерв
- общо

#### `components/SummaryCard.jsx`
Показва:
- общ брой стаи
- общо осветление
- общо контакти
- общо консуматори
- общ резерв
- общ кабел за всички стаи

#### `utils/calculateCable.js`
Съдържа логиката за ориентировъчно изчисляване на кабела.

Във версия 1.1 резултатът вече е структуриран като обект:

```js
result: {
  lighting,
  sockets,
  appliances,
  reserve,
  total
}
```

#### `utils/storage.js`
Съдържа помощни функции за работа с `localStorage`.

Добавена е нормализация на данните при зареждане, за да не се чупи приложението при:
- стари записи
- различна структура на `result`
- липсващи стойности

## Логика на изчислението

В текущата версия формулата е **ориентировъчна**, а не проектантска.

Изчислението използва:

- размери на стаята
- периметър на помещението
- средна хоризонтална дължина
- вертикални трасета според начина на полагане
- добавки според източника на трасето
- отделно изчисление за:
  - осветление
  - контакти
  - консуматори
- резервен процент

Целта на тази формула е да даде **практична начална сметка**, без сложна схема и без точно позициониране на точките.

## Начални стойности във формата

За удобство са зададени следните стойности по подразбиране:

- височина на контактите: `0.30 м`
- височина на ключовете: `1.20 м`
- източник на трасето: `разклонителна кутия`
- начин на полагане: `по таван`
- резерв: `10%`

## Стабилност на данните

При преминаване от по-стара версия към новата структура на резултатите беше добавена защита в:

- `storage.js`
- `RoomList.jsx`
- `SummaryCard.jsx`

Това позволява:

- старите записи да бъдат нормализирани
- липсващите стойности да се третират като `0`
- приложението да не показва `NaN`
- интерфейсът да остане стабилен при бъдещи промени
